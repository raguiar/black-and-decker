Extração de dados num clique.

Sumariza num ficheiro excel os dados obtidos de um conjunto de ficheiros excel com um formato comum. A pasta dos ficheiros, localização das células com os dados e ficheiro de destino são configurados num ficheiro 'config.xlsx' que deve residir ao lado deste programa.
